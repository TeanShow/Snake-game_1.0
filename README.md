# 🐍 Snake Game 🎮

Welcome to the classic Snake Game with a modern twist! This implementation brings back the nostalgia of the old mobile phone game with enhanced features.

## ✨ Features

- 🌃 **Dark Neon Theme** - Stylish dark interface with vibrant neon colors for an immersive gaming experience
- 🏆 **High Score Tracking** - Your best scores are saved using LocalStorage so you can beat your personal records
- ⚡ **Increasing Difficulty** - The snake moves faster as you collect more food, making the game progressively more challenging
